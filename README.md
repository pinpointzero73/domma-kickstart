# Domma Kickstart: Your Starting Point for Modern Web Apps

**A lightweight, zero-dependency kickstart repository to get you building modern web applications with the [Domma](http://domma.dcbw-it.co.uk/) framework.**

This template provides a simple and clean starting point for your next project. It includes a pre-configured `index.html` file with the Domma framework, a sample application structure, and a guide to get you up and running in minutes.

## Getting Started

To get started, simply clone this repository and open the `index.html` file in your browser.

```bash
git clone https://github.com/your-username/domma-kickstart.git
cd domma-kickstart
# Open index.html in your browser
```

The main application logic is located in `includes/index.js`. You can start editing this file to build your application.

## Features

This kickstart template is designed to showcase the power of the Domma framework, which includes:

*   **Zero Dependencies:** The entire framework is contained in a single `domma.min.js` file.
*   **jQuery-Compatible API:** A familiar and intuitive API for DOM manipulation.
*   **Lodash-Compatible Utilities:** Over 120 utility functions for data manipulation.
*   **Rich Component Library:** A wide range of pre-built UI components, including modals, tabs, carousels, and more.
*   **Responsive Grid System:** A flexible 12-column grid system for building responsive layouts.
*   **Theming:** A built-in theme system with light/dark modes and multiple color variants.
